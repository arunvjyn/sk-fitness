# sk-fitness
Official website for SK Fitness, a professional gym located in Puducherry, built to showcase training, facilities, certifications, and provide easy contact for members. This project is a modern, responsive website designed to deliver a clean user experience and represent the disciplined fitness culture of SK Fitness.
# 🏋️ SK Fitness – Official Website

This is the official website for **SK Fitness**, a professional gym located in Puducherry.

## 📌 About the Project

This website is designed to:
- Showcase the gym environment and equipment  
- Highlight certifications and professionalism  
- Provide contact details for potential members  
- Represent the disciplined fitness culture of SK Fitness
